# Sistemas-de-Capatacion-de-Lluvia-Ciudad-de-Mexico-2019-a-2024

**Lenguaje: Python 3.x**

**Librerías: Pandas, Matplotlib, Seaborn, NumPy**

**Entorno: Google Colab / Jupyter Notebook**


## Este proyecto analiza la distribución geográfica y el impacto del programa de captación pluvial en la Ciudad de México. El objetivo es identificar las zonas con mayor adopción tecnológica y entender la cobertura del programa frente a las necesidades hídricas de la capital. Se realizó la limpieza de coordenadas y corrección de registros atípicos (fuera de las coordenadas de la Ciudad de México) para asegurar mapas precisos.
